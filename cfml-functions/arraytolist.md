# ArrayToList

### Example using member function

```coldfusion
<cfscript>
// Declare an array
myArray=["Google","Microsoft","Adobe","Facebook","Amazon"];
myConvertedList=myArray.toList();
WriteOutput(myConvertedList);
</cfscript>
```
### Output
Google,Microsoft,Adobe,Facebook,Amazon
